# gba-template

## Install

- Install `devKitPro` and clone the project. Use this file structure:
	* `gba`
		* `tools`
			* `devKitPro`
		* `projects`
			* `gba-template`
- Configure environment variables:
	* `PATH` (add `{DEVKITARM}/bin` and `{DEVKITPRO}/tools/bin`)
- Run script:
	* `./configure.sh`
- Never move any folder ¯\_(ツ)_/¯

## Actions

- `make clean`
- `make build`
- `make start`
